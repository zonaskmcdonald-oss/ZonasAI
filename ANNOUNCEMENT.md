ZonasAI — v0.1.0 Release
=========================

What's included
----------------
- Simple "Hello World" examples for Python, JavaScript, TypeScript, and Go.
- A lightweight Node/Express web server to run examples via HTTP endpoints.
- `package.json` and `tsconfig.json` to run TypeScript examples.
- `render.yaml` manifest for one-click deploy to Render.

How to try it (quick)
---------------------
1. Clone the repo: `git clone https://github.com/zonaskmcdonald-oss/ZonasAI.git`
2. Install Node deps: `npm install`
3. Start the server: `npm start`
4. Open: `http://<host>:3000/run?lang=python&name=Alice`

Deploy
------
This release includes `render.yaml` so you can deploy to Render with automatic detection. See README for full steps.
