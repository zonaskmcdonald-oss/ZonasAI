This folder contains simple "Hello World" examples for multiple languages.

Languages included:

- Python: `examples/python/hello.py`
- JavaScript: `examples/javascript/hello.js`
- TypeScript: `examples/typescript/hello.ts`
- Go: `examples/go/hello.go`

Run examples (requirements listed per language):

Python
```
python3 examples/python/hello.py Alice
```

JavaScript (Node.js)
```
node examples/javascript/hello.js Alice
```

TypeScript
```
# Use ts-node (install with `npm i -D ts-node typescript`) or compile with `tsc`
npx ts-node examples/typescript/hello.ts Alice
```

Go
```
go run examples/go/hello.go Alice
```
